一方
ruby make_individual.rb --pedir pe
とすると、pe/を見に行ってquote.html.erbにヒストグラム情報を流してHTML化という流れ

関連銘柄リスト作成は
./analyze -cor -cordir cor
とやると、
cor/以下にどかっとcor/8031という具合に関連銘柄ベスト10が出力される。

こいつを
ruby make_individual.rb --cordir cor
とやって取り込む。
